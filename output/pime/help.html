小麥注音輸入法 PIME 版本輔助說明

常見問題

問：PIME 版本與 macOS、Linux 版本有什麼差別？

答：PIME 版本是基於小麥注音的 JavaScript 版本，也就是與網頁版本以及 Chrome OS 版本共用了大量的程式碼。因為每個平台能夠支援的系統能力不一樣，所以也都有一些不同的地方。不過，選字引擎、詞庫、按鍵處理邏輯等，與 macOS/Linux 版本是相同的。

問：小麥注音會使用多少系統資源？

答：小麥注音啟動之後，會佔用 7MB 左右的詞庫，啟動時間大約 0.7 秒左右，接著整個系統都共用這份詞庫。跟現在許多的網頁應用相較，小麥注音佔用的資源並不算多。

一般輸入

小麥注音如同大多數的注音輸入法，您只需要輸入注音符號按鍵，然後按下代表四聲的按鍵，就會組合出中文，按下空白鍵可以選擇候選字。除了標準鍵盤配置外，小麥還支援倚天、許氏、倚天26鍵等鍵盤配置，您可以在小麥的選項中修改配置。

標點輸入

小麥注音在標準鍵盤配置下，輸入標點符號的按鍵如下：

此外，如果按下鍵盤上的「`」按鈕，也會出現一個簡單的符號表。

特殊符號輸入

小麥支援一輸入一些特殊符號：

  • 像是在輸入國家名稱的時候，也可以從選字中，選擇國旗，像是輸入美國，就可以選擇「🇺🇸」。
  • 一些動物名稱也會有對應的符號，例如打「龜」可以選擇「🐢」，打「鳥」可以選擇「🐦」。
  • 其他：像是選「火」可以選擇「🔥」。

加入自訂詞彙

在打字的時候，如果輸入緩衝區當中有文字內容，您可以按住 Shift，然後用左右鍵選出一定的範圍，然後按下Enter，就可以將選擇的範圍變成新的自訂詞彙。不過，如果輸入法本身的詞彙就已經包含這個詞彙，或是您之前已經加過這個詞，就無法加入。

一個自訂詞彙最少需要兩個字，最多六個字。小麥注音會將自訂詞儲存到您本機的一個純文字檔案中。

您也可以在「管理使用者自訂詞」功能中，看到所有您加入的自訂詞。在這個畫面中，您可以看到中央的文字編輯框中，裡頭包含了您目前所有的自訂詞，您可以在這邊自由新增、刪除自訂詞，格式是「文字 注音」，像是「自訂詞ㄗˋ-ㄉㄧㄥˋ-ㄘˊ」； 編輯完成後，按下儲存即可。

簡繁轉換

小麥注音支援使用注音輸入法輸入簡體中文。您可以從輸入法選單中，點選「使用簡體中文」選項切換。另外，按下Shift + Ctrl + g，可以快速切換簡繁轉換。

特殊輸入功能

小麥注音還提供以下輸入功能

  • 當您輸入「今天」時，可以快速輸入今天日期，像是「民國113年8月6日」。打「明天」、「昨天」也有同樣的功能。
  • 當您輸入「今年」時，可以快速輸入今年是哪一年，像是「2024年」、「民國113年」等。打「去年」、「明年」也有一樣的功能。
  • 按下 Ctrl + ＼，可以出現一個輸入功能選單，當中的「日期與時間」，就可以選擇今天的日期或年份。
  • 輸入功能選單中的「中文數字」，可以讓您輸入「1234」，快速轉換成「一千二百三十四」。
  • 輸入功能選單中的「大寫數字」，可以讓您「1234」，快速轉換成「壹仟貳佰參拾肆」。
  • 至於蘇州碼，則是一種比較古老的中文數字系統，但是在一些地區仍然在使用中。例如，如果想輸入「1234元」，輸出結果如下：

〡二〣〤
千元

字典查詢

在出現選字窗的時候，或是用 Shift + 左右按鍵，選擇出一段文字時，可以按下問號（?）按鍵，就會出現一個字典選單，可以用萌典等字典查詢指定的候選字。

選項

小麥注音支援以下選項：

  • 鍵盤配置：大部分人大概會選擇標準配置，不過，曾經用過 倚天中文的用戶，可能會選擇倚天排列或是倚天 26 鍵排列；曾經用過自然輸入法 的用戶，可能會選擇許氏鍵盤。小麥另外支援漢語拼音以及IBM 鍵盤 排列。
  • 選字按鍵：在使用空白鍵叫出選字窗之後，我們大多使用數字選字，不過小麥注音也提供其他的選字按鍵選項。
  • 前選字與候選字模式：這個選項代表的是，當您在按下空白鍵選字的時候，是要讓游標之前，還是游標之後的文字進入選字狀態。如果您之前使用的是 macOS 或習慣漢音輸入法，那麼選擇往前選字會比較符合您的習慣。如果之前用的是自然、微軟新注音等輸入法，您可以選擇向後選字。
  • 選字後自動移動游標位置：這是一項提供給習慣微軟新注音輸入法的選項。因為選字模式是後選字模式，所以，往往是打了一些字之後，我們會往前移動游標選字，這個選項打開後，就可以在選字後，自動移動回之前打字的位置。
  • ESC 按鍵清除整個輸入緩衝區：這也是一項符合微軟新注音輸入法習慣的選項。小麥的預設行為是，在按下 ESC 之後，只會清除還沒有組字完成的注音，但開啟這個選項後，會清除整的輸入緩衝區的內容。
  • 其他：您可以選擇輸入Shift + 字母按鍵，要輸入的是大小或是小寫的英文字母。
