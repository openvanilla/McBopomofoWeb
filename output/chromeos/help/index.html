小麥注音輸入法 Chrome OS 版本輔助說明

常見問題

問：為什麼往往切到小麥注音開始打字的時候，會先打出英文，而不是中 文？

答：小麥注音使用 Google Chrome Extension Manifest V3 開發，在您打字的時 候，Chrome OS 系統需要先啟動小麥注音的背景 worker，小麥注音才有辦法開始處理您 所輸入的鍵盤按鍵。不過，Chrome OS大概一定時間，就會讓我們的 Worker 停止運作， 所以，下次開始打字時，又得重新開始背景 Worker，又會出現想打中文卻跑出英文的 狀況。

問：是否提供橫向選字窗？

答：Chrome OS 的確提供橫向選字窗，不過經過測試，這個橫向選字窗的功 能有些問題，所以我們目前只提供垂直的選字窗介面。

問：Chrome OS 版本與 macOS、Linux 版本有什麼差別？

答：Google 允許我們使用網頁相關技術如 HTML、JavaScript 等，在 Chrome OS 上製作輸入法，所以 Chrome OS 上，我們將原本的輸入邏輯，整個搬移到 網頁相關的技術上。但選字引擎、詞庫、按鍵處理邏輯等，與 macOS/Linux 版本是相 同的。而由於Chrome OS 系統中大量使用 Ctrl 的系統快速鍵，所以我們也移除了一些 原本使用 Ctrl 按鈕輸出的標點符號。

問：小麥注音會使用多少系統資源？

答：小麥注音啟動之後，會佔用 7MB 左右的詞庫，接著整個系統都共用這 份詞庫。跟現在許多的網頁應用相較，小麥注音佔用的資源並不算多。

安裝與啟用

您可以直接在 Google 的 Chrome 線上應用程式商店中，前往 小麥注音的頁面，選擇「加入到 Chrome」，便可以安裝小麥注 音。

安裝完畢後，請到 Chrome OS 的系統選項中，選擇「裝 置」->「鍵盤」->「變更輸入法設定」；或是「進階」->「語言和輸入 法」->「輸入法」中，選擇「新增輸入法」，從裡頭挑選小麥注音。之後，您 使用 Ctrl + Space 切換輸入法時，就可以切換到小麥注音了。

如果您想要刪除小麥注音，可以前往 Chrome 的延伸功能管理功能 （chrome://extensions），找到小麥注音，然後選擇刪除。

一般輸入

小麥注音如同大多數的注音輸入法，您只需要輸入注音符號按鍵，然後按下代表四 聲的按鍵，就會組合出中文，按下空白鍵可以選擇候選字。除了標準鍵盤配置 外，小麥還支援倚天、許氏、倚天26鍵等鍵盤配置，您可以在小麥的選項中修改配置。

特殊符號輸入

小麥支援一輸入一些特殊符號：

  • 像是在輸入國家名稱的時候，也可以從選字中，選擇國旗，像是輸入美國，就可以選擇「🇺🇸」。
  • 一些動物名稱也會有對應的符號，例如打「龜」可以選擇「🐢」，打「鳥」可以選擇「🐦」。
  • 其他：像是選「火」可以選擇「🔥」。

加入自訂詞彙

在打字的時候，如果輸入緩衝區當中有文字內容，您可以按住 Shift，然後用左右 鍵選出一定的範圍，然後按下 Enter，就可以將選擇的範圍變成新的自訂詞彙。不過， 如果輸入法本身的詞彙就已經包含這個詞彙，或是您之前已經加過這個詞，就無法加 入。

一個自訂詞彙最少需要兩個字，最多六個字。小麥注音會將自訂詞儲存到您的 Sync Storage 中，所以，當您換到另外一台Chrome OS 裝置時，這些自訂詞也會一同搬移到 另一台裝置上。

您也可以在「管理使用者自訂詞」功能中，看到所有您加入的自訂詞。在這個畫面 中，您可以看到中央的文字編輯框中，裡頭包含了您目前所有的自訂詞，您可以在這邊 自由新增、刪除自訂詞，格式是「文字 注音」，像是「自訂詞 ㄗˋ-ㄉㄧㄥˋ-ㄘˊ」； 編輯完成後，按下儲存即可。

簡繁轉換

小麥注音支援使用注音輸入法輸入簡體中文。您有兩種方式可以切換檢繁體輸入：

  • 在 Chrome OS 的選單中，選擇鍵盤，然後選擇小麥注音，然後就可以看到小 麥注音的選單，從中勾選「使用簡體中文輸入」即可。
  • 您也可以透過 Chrome 的快速鍵設定（chrome://extensions/shortcuts）， 幫簡繁轉換設定一組快速鍵，之後只要按下這組快速鍵，就可以切換簡繁中文 輸入。

特殊輸入功能

小麥注音還提供以下輸入功能

  • 當您輸入「今天」時，可以快速輸入今天日期，像是「民國113年8月6日」。打「明天」、「昨天」也有同樣的功能。
  • 當您輸入「今年」時，可以快速輸入今年是哪一年，像是「2024年」、「民國113年」等。打「去年」、「明年」也有一樣的功能。
  • 按下 Ctrl + ＼，可以出現一個輸入功能選單，當中的「日期與時間」，就可以選擇今天的日期或年份。
  • 輸入功能選單中的「中文數字」，可以讓您輸入「1234」，快速轉換成「一千二百三十四」。
  • 輸入功能選單中的「大寫數字」，可以讓您「1234」，快速轉換成「壹仟貳佰參拾肆」。
  • 至於蘇州碼，則是一種比較古老的中文數字系統，但是在一些地區仍然在使用中。例如，如果想輸入「1234元」，輸出結果如下：

〡二〣〤
千元

選項

小麥注音支援以下選項：

  • 鍵盤配置：大部分人大概會選擇標準配置，不過，曾經用過 倚天中文的用戶，可能會選擇倚天排列或是倚天 26 鍵排列；曾經用 過自然輸入法 的用戶，可能會選擇許氏鍵盤。小麥另外支援漢語拼音以及IBM 鍵盤 排列。
  • 選字按鍵：在使用空白鍵叫出選字窗之後，我們大多使用數字選字， 不過小麥注音也提供其他的選字按鍵選項。
  • 前選字與候選字模式：這個選項代表的是，當您在按下空白鍵選字的 時候，是要讓游標之前，還是游標之後的文字進入選字狀態。如果您之前使用的是 macOS或習慣漢音輸入法，那麼選擇往前選字會比較符合您的習慣。如果之前用的 是自然、微軟新注音等輸入法，您可以選擇向後選字。
  • 選字後自動移動游標位置：這是一項提供給習慣微軟新注音輸入法的 選項。因為選字模式是後選字模式，所以，往往是打了一些字之後，我們會往前移 動游標選字，這個選項打開後，就可以在選字後，自動移動回之前打字的位 置。
  • ESC 按鍵清除整個輸入緩衝區：這也是一項符合微軟新注音輸入法習 慣的選項。小麥的預設行為是，在按下 ESC 之後，只會清除還沒有組字完成的注 音，但開啟這個選項後，會清除整的輸入緩衝區的內容。
  • 其他：您可以選擇輸入 Shift + 字母按鍵，要輸入的是大小或是小寫的 英文字母。
